# Installation

## Prerequisites

- Python 3.11 or higher
- pip package manager

## Install from PyPI

```bash
pip install codex-aura
```

## Install from source

```bash
git clone https://github.com/codex-aura/codex-aura.git
cd codex-aura
pip install -e .
```

## Development installation

For development with all dependencies:

```bash
pip install -e ".[dev]"
```

## Verify installation

```bash
codex-aura --version
```

You should see the version number if installation was successful.
