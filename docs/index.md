# Welcome to Codex Aura

Codex Aura is a powerful code analysis tool that generates dependency graphs and provides insights into your codebase structure.

## What is Codex Aura?

Codex Aura analyzes your Python codebase and creates a comprehensive dependency graph showing relationships between modules, classes, and functions. It helps you understand:

- Import relationships
- Function calls
- Class inheritance
- Code complexity metrics

## Quick Start

Get started with Codex Aura in just a few steps:

1. Install the package
2. Analyze your codebase
3. Explore the generated graph

## Features

- **Dependency Analysis**: Automatically detect and visualize code dependencies
- **Multiple Output Formats**: JSON, GraphML, and interactive web interface
- **CLI Tools**: Command-line interface for easy integration
- **API Server**: REST API for programmatic access
- **GitHub Action**: Integrate analysis into your CI/CD pipeline
- **VS Code Extension**: Visual analysis directly in your editor
