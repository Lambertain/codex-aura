# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-20

### Added
- Initial release
- Python analyzer with IMPORTS support
- CLI tool with `analyze` command
- JSON graph format v0.1
- Comprehensive test suite with 84%+ coverage
- Documentation (README, API docs, contributing guide)
- CI/CD pipeline with GitHub Actions
- Support for relative imports and complex project structures
- Edge case handling (empty repos, syntax errors, large files)
- Performance benchmarks for real-world repositories
